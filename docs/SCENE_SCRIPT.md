**Scene-by-Scene Story Script — Home, Senders, Travelers, Trust,
Company**

*Companion to the Website Storytelling Plan \| Phase 0 deliverable —
lock this before Figma or code*

How to Use This Script

This reads like a shot list for a short film, not marketing copy. Each
row is one scroll-driven "scene" — roughly one screen-height of scroll
on desktop. Copy is deliberately short: these are subtitles the visitor
reads in 2–3 seconds while something moves, not paragraphs to sit and
read.

- Recurring characters: "the Traveler" (our guide, reintroduced in
  Home/Travelers) and rotating Senders (different people per scene to
  show breadth of use cases).

- Named personas below (Arjun, Priya) are placeholders for your
  storyboard/copy team — swap for whatever names or illustrated look you
  land on.

- Treat the last scene of every chapter as a soft "chapter cut" — a
  brief transition (color wash / shape wipe) into the next chapter, not
  a hard page load.

Notes on Contact & FAQ

These stay deliberately outside the cinematic script above — per the
original vision, they're lightweight utility pages using the same design
system (typography, color, icon set) but with no scroll-driven scenes: a
simple form on Contact, a plain accordion on FAQ.

What's Next
